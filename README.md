# Dashboard GPMB — Tổ công tác số 4

Dashboard HTML tĩnh, không cần cài đặt thư viện hoặc máy chủ.

## Chạy

- Mở trực tiếp `index.html` bằng trình duyệt; hoặc
- Trong thư mục này, chạy một máy chủ web tĩnh bất kỳ rồi mở trang chủ.

## Cấu trúc

- `index.html`: khung nội dung và siêu dữ liệu.
- `styles.css`: giao diện responsive desktop/mobile.
- `app.js`: dữ liệu dự án, KPI, biểu đồ, bộ lọc và thẻ chi tiết.
